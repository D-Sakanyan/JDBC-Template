# План ручного тестирования SkillFitBox через UI

## Обзор

Здесь вы найдете детальный план ручного тестирования пользовательского интерфейса приложения SkillFitBox - системы управления фитнес-центром. Тестирование проводится через веб-интерфейс, доступный по адресу `http://localhost:8090`.

## Подготовка к тестированию

### Предварительные условия
1. Приложение запущено и доступно по адресу `http://localhost:8090`
2. База данных PostgreSQL настроена и содержит начальные данные
3. Браузер поддерживает современные веб-стандарты
4. JavaScript включен в браузере

### Начальные данные
- 20 шкафчиков (номера 1-20)
- 5 дополнительных услуг: Солярий, Бассейн, Сауна, Криосауна, Кроссфит
- Пустые таблицы клиентов и тренеров

## 1. Тестирование управления клиентами

### 1.1 Просмотр списка клиентов
**Цель:** Проверить отображение списка клиентов

**Шаги:**
1. Перейти в раздел "Клиенты"
2. Проверить отображение списка

**Ожидаемый результат:**
- Отображается заголовок "Клиенты"
- Если клиентов нет, показывается сообщение "Клиенты не найдены"
- Присутствует кнопка "Добавить клиента"

### 1.2 Создание нового клиента
**Цель:** Проверить функциональность создания клиента

**Шаги:**
1. На странице списка клиентов нажать "Добавить клиента"
2. Заполнить форму:
   - Фамилия: "Иванов"
   - Имя: "Иван"
   - Отчество: "Иванович"
   - Дата рождения: "1990-05-15"
   - Телефон: "+79123456789"
   - Email: "ivan.ivanov@example.com"
   - Статус: "Активен" (по умолчанию)
3. Нажать "Создать клиента"

**Ожидаемый результат:**
- Клиент успешно создается
- Появляется сообщение об успехе
- Происходит перенаправление на страницу списка клиентов
- Новый клиент отображается в списке

### 1.3 Просмотр детальной информации о клиенте
**Цель:** Проверить отображение детальной информации

**Шаги:**
1. В списке клиентов кликнуть на имя клиента
2. Проверить отображение детальной информации

**Ожидаемый результат:**
- Отображается полная информация о клиенте
- Присутствуют кнопки "Редактировать", "Назначить тренера", "Назначить шкафчик", "Добавить услугу"
- Если тренер/шкафчик/услуги не назначены, отображается соответствующая информация

### 1.4 Редактирование клиента
**Цель:** Проверить функциональность редактирования

**Шаги:**
1. На странице клиента нажать "Редактировать"
2. Изменить email на "ivan.ivanov.updated@example.com"
3. Нажать "Обновить клиента"

**Ожидаемый результат:**
- Изменения сохраняются
- Появляется сообщение об успехе
- Обновленная информация отображается на странице клиента

### 1.5 Изменение статуса клиента
**Цель:** Проверить функциональность изменения статуса

**Шаги:**
1. На странице клиента нажать кнопку изменения статуса
2. Подтвердить изменение

**Ожидаемый результат:**
- Статус клиента изменяется
- Появляется сообщение об успехе
- Обновленный статус отображается на странице

## 3. Тестирование управления шкафчиками

### 3.1 Просмотр списка шкафчиков
**Цель:** Проверить отображение списка шкафчиков

**Шаги:**
1. Перейти в раздел "Шкафчики"
2. Проверить отображение списка

**Ожидаемый результат:**
- Отображается заголовок "Шкафчики"
- Показываются все 20 шкафчиков
- Для каждого шкафчика отображается номер и статус (Свободен/Занят)
- Присутствует статистика: общее количество, свободные, занятые

### 3.2 Проверка статусов шкафчиков
**Цель:** Проверить корректность отображения статусов

**Шаги:**
1. На странице шкафчиков проверить статусы
2. Назначить клиенту шкафчик (через раздел клиентов)
3. Вернуться на страницу шкафчиков
4. Проверить изменение статуса

**Ожидаемый результат:**
- Свободные шкафчики показывают статус "Свободен"
- Занятые шкафчики показывают статус "Занят" и имя клиента
- Статистика обновляется корректно
